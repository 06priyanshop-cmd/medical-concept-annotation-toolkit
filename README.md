# Medical Concept Annotation Toolkit

An AI-powered Medical Concept Annotation Toolkit built using Flask, MedCAT, MySQL, and NLP.

## Features

- Medical entity extraction using MedCAT
- Clinical text highlighting
- Manual entity creation
- CDB search
- SQL entity storage
- Entity statistics
- CSV export

## Technologies

- Python
- Flask
- MedCAT
- MySQL
- HTML
- CSS
- JavaScript
